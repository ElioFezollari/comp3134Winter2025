1) What is the definition of server port?

The answer is a server port is a form of unique "endpoint" used to uniquely tell from different services at which data is sent

2) What is server port range?

The answer is, the server port range is a range of numbers from 0 to 65535, which a server can use to communicate over a network.
The reason for such a weird number is because its a 16 bit integer and 2^16 is 65535

3) How many bits does number above represent?
The answer is 16 bits as  2^16 = 65535
